Puppy DB Tool
GitHub Repo Latest Release

Desktop Database Manager

Connect to MySQL, PostgreSQL, MongoDB, and Redis. Use direct connection or SSH tunnel, browse tables, edit rows, and run queries from a modern dark UI.

How To Use

1. Add Connection
Create direct or SSH tunnel connection.
2. Open Database
Browse schemas, tables, and structure metadata.
3. Manage Data
Insert, update, delete, filter, and sort rows.
4. Run SQL/Commands
Use Query Studio with snippets and history.
5. Export
Export full database and query results.
6. Table Actions
Browse, structure, empty, drop, and create tables.
